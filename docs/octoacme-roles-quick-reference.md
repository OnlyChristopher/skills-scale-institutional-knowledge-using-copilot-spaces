## Decision Authority

### Product Decisions
- **Final Authority:** Product Manager
- **Input from:** Stakeholder Liaison, Developers, Project Manager

### Technical Decisions
- **Final Authority:** Developer (team consensus)
- **Input from:** QA Lead, Product Manager, Project Manager

### Process Decisions
- **Final Authority:** Process Owner
- **Input from:** Project Manager, all roles

### Quality Standards
- **Final Authority:** QA Lead
- **Input from:** Process Owner, Developers

### Schedule & Resources
- **Final Authority:** Project Manager
- **Input from:** Developers, Product Manager

### Scope Changes
- **Final Authority:** Product Manager (with stakeholder approval)
- **Facilitated by:** Change Manager
- **Input from:** Project Manager, Developers
